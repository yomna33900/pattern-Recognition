# BMW Car Sales Classifier

A machine learning project that classifies BMW car sales 
performance using pattern recognition techniques.

## Description
Predicts and classifies BMW car sales using multiple ML models
with feature selection to improve accuracy.

## Features
- Data preprocessing and cleaning
- Feature selection using SelectKBest & Mutual Information
- Multiple models: Random Forest & XGBoost
- Accuracy comparison before and after feature selection
- Data visualization with Seaborn & Matplotlib

## Technologies Used
- Python
- Random Forest & XGBoost
- Scikit-learn
- Pandas & NumPy
- Seaborn & Matplotlib
